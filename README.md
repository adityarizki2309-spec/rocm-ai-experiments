# AMD ROCm LLM Benchmark

Open-source experiments for AI inference and benchmarking on AMD GPUs using ROCm.

## Goals

- Test AMD GPU compatibility with AI frameworks
- Benchmark LLM inference performance
- Evaluate ROCm optimization opportunities
- Share reproducible results with the community

## Planned Features

- PyTorch ROCm benchmarks
- LLM inference examples
- Performance comparison reports
- GPU utilization monitoring

## Status

Work in progress.
